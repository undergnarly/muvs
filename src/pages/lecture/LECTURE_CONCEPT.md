**Заголовок:** Как говорить с AI
**Формула идеального промпта:**
```
Роль + Контекст + Задача + Ограничения + Формат
```
**Пример:**
"Ты - эксперт по продуктивности (роль). Мне нужно организовать рабочий день (контекст). Предложи расписание с учётом моих биоритмов (задача). Только 3 приоритета, остальное - делегировать (ограничения). В виде таблицы (формат)."

## Визуальный стиль

### Цветовая палитра
- **Фон:** Тёмный минимализм (#000000 - #0a0a0a)
- **Акцент 1:** Энергетический красный (#d32f2f) - символ Шаолиня
- **Акцент 2:** Технологический синий (#2196f3) - символ AI
- **Текст:** Белый (#ffffff) и светло-серый (#b0b0b0)
- **Градиенты:** Плавные переходы между акцентами

### Типографика
- **Заголовки:** Bold, geometric sans-serif
- **Текст:** Regular, readable sans-serif
- **Размеры:** Крупные заголовки, читаемый основной текст
- **Иерархия:** Чёткое разделение уровней

### Композиция
- **Центрированная:** Для заголовков и ключевых идей
- **Сеточная:** Для списков и сравнений
- **Минималистичная:** Максимум воздуха, минимум шума
- **Фокус:** Одна главная мысль на слайд

### Анимации
- **Фейды:** Плавное появление
- **Слайды:** Легкие transitions
- **Акценты:** Subtle highlighting
- **Без лишнего:** Никаких distractings
