# Лекция "Автоматизация и AI системы"

## Описание
Интерактивная презентация для Шаолинь Кэмпа на тему автоматизации и AI-систем. Презентация помогает внедрить AI в повседневную жизнь и организовать личную систему продуктивности.

## Доступ к лекции
- **URL:** `/lecture`
- **Полный URL:** `http://localhost:5173/lecture` (для локальной разработки)

## Управление презентацией

### Клавиатурные горячие клавиши
- **→** или **Пробел** - следующий слайд
- **←** - предыдущий слайд
- **F** - переключить полноэкранный режим
- **Escape** - выйти из полноэкранного режима или вернуться на главную страницу
- **0-9** - быстрая навигация к слайдам 1-9

### Мышь/Тач
- Стрелки на экране - навигация между слайдами
- Индикаторы внизу - быстрый переход к любому слайду
- Кнопка Home (слева вверху) - выход на главную страницу
- Кнопка полноэкранного режима (вторая слева вверху) - включить/выключить полноэкранный режим

## Использование

### Для спикера
1. Откройте `/lecture`

3. Используйте стрелки или пробел для навигации
4. Следите за индикатором времени на каждом слайде
5. Используйте Escape для быстрого выхода

```bash

# Сборка для продакшена

# Предпросмотр сборки
```

## Модификация контента

### Изменение слайда
Откройте `LecturePage.jsx` и найдите массив `lectureSlides`. Каждый слайд - это объект со следующей структурой:

```javascript
{
  block: 'intro',      // Блок (intro, fundamentals, everyday, automation, future)
  id: 1,               // Номер слайда
  title: 'Заголовок',  // Заголовок слайда
  content: {...},      // Контент (зависит от типа)
  type: 'title',       // Тип слайда
  timing: '3 мин'      // Рекомендуемое время (опционально)
}
```

### Типы слайдов
- `title` - Титульный слайд
- `list` - Список
- `balance` - Баланс (Шаолинь + AI)
- `checklist` - Чек-лист
- `bio` - Биография
- `concept` - Концепция
- `process` - Процесс (шаги)
- `tools` - Инструменты
- `formula` - Формула
- `warning` - Предупреждение
- `exercise` - Практическое упражнение
- `routine` - Рутина
- `learning` - Обучение
- `wellness` - Здоровье
- `creative` - Творчество
- `communication` - Коммуникации
- `finance` - Финансы
- `automation-examples` - Примеры автоматизации
- `system` - Система (PARA)
- `knowledge` - Знания
- `trends` - Тренды
- `skills` - Навыки
- `ethics` - Этика
- `plan` - План
- `resources` - Ресурсы
- `final` - Финальный слайд
